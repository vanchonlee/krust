# Contributing to Krust (Public Repo)

Thanks for helping improve Krust.

This repository is used for:

- bug reports
- feature requests
- product feedback
- docs and website feedback

This repository is **not** used for application source code pull requests.

## Before opening an issue

1. Search existing issues first.
2. Use the appropriate issue template.
3. Add clear reproduction steps and environment details.

## Bug report checklist

- Krust version
- macOS version
- Kubernetes cluster version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots/logs if available

## Security reports

Please do not post security vulnerabilities publicly.
Email: **security@krust.io**
